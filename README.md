# Calculator

A simple calculator built with HTML, CSS, and JavaScript.
The UI is inspired by the Infinix Smart 8 Pro built-in calculator.

## Features

* Basic operations
* Percentage support
* Delete and clear functions
* Keyboard support
* Error handling for divide by zero

## How to Use

Click the buttons or use your keyboard.
Press Enter for result, Backspace to delete, and Escape to clear.
